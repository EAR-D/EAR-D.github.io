搭建博客

需要的软件
Git
node.js

首先使用淘宝的镜像源（错误就重来，连接有时不稳定，多试几次）
npm install -g cnpm –registry=https://registry.npm.taobao.org

检测

cnmp

cnmp -v

安装hexo(失败继续重来)
cnpm install -g hexo-cl
检测
hexo -v

cd D:/hexo/blog (进入blog文件夹)

pwd(检测)

hexo init(初始化hexo)

hexo s (启动hexo)

INFO Hexo is running at http://localhost:4000 .Press Ctrl+C to stop.
看到这句就基本成功了

在网页上输入http://localhost:4000

第二章

hexo n”你想要的标题”

检测

ls -l

进入vim

vim +(标题)

第三章
vim 一些基础

编辑结束c时 按Esc 之后 “:”wq
强退 “:”q!
